# Segmented component

### Usage

```
import SegmentedControl from './components/segmented-control';

<SegmentedControl
  shade="light"
  onChange={this.onOptionsChanged}
  options={[
    { label: 'Option 1', value: 'option-1' },
    { label: 'Option 2', value: 'option-2' },
    { label: 'Option 3', value: 'option-3' },
  ]}
/>
```
